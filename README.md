### requirements
* [mod.io API Key](https://mod.io/me/access)
* windows 10/11 

### install (dev)
1.  ``git clone https://github.com/aaesth/boneMM``
2. ``npm install``
3. ``npm start``

### building the exe
``
npm run build:win
``
output will be in the /dist
